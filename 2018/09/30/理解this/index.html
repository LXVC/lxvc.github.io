前言

this 的指向问题在 JavaScript 中对初学者来说一直就像一个谜一样。

在其他面向对象语言中，this 关键字只有在对象调用自身的属性或者方法时才会出现，此时 this 就是指向调用者的一个指针，这很符合直觉，但在 JavaScript 中就复杂很多。

在我看来，JavaScript 中的 this 分为两大类：

  1. 作为函数调用，指向宿主环境的全局对象
  2. 当作方法调用，this 指向调用它的对象

1. 作为函数调用时

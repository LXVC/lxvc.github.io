is 实质上是对变量进行 id() 求值后的返回值比较，在对值类型的变量进行比较时用 is 还是 ==，没有区别；在对对象实例进行比较时，如果需要判断两个变量是否是同一个实例的引用，用 is，如果需要判断两个对象实例变量的值是否完全相等，用 ==。特别的：如果 a == b 为 True，a is b 不一定为 True；如果 a is b 为 True，则 a == b 一定为 True
